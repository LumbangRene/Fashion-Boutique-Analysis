# Fashion Boutique Excel Analysis

A comprehensive Excel project analyzing sales, returns, discounts, and revenue for a fashion boutique. This project combines data cleaning, formulas, and pivot tables to extract actionable insights while reflecting my passion for fashion and analytics.

## Project Overview

This dataset contains product-level sales data including:

- **Product details:** category, brand, season, size, color
- **Pricing:** original price, markdown percentage, current price, calculated discounts
- **Sales data:** purchase date, stock quantity, customer ratings
- **Returns:** returned items and reasons

The project demonstrates how to transform raw data into meaningful insights.

## Key Features

- **Data Cleaning & Validation**
  - Ensured logical consistency (e.g., return status matches reason)
  - Checked pricing accuracy using formulas
  - Handled blanks and missing values effectively

- **Formulas & Calculations**
  - `IF` statements for data validation
  - Calculated amount discounted and price checks
  - Revenue calculations per product

- **Pivot Tables & Analysis**
  - Returns by brand, category, and season
  - Average customer rating per category and brand
  - Total revenue by brand and seasonal revenue analysis
  - Markdown impact on returns

- **Charts & Visual Insights**
  - Bar and column charts to visualize returns and revenue
  - Helps identify trends in customer behavior and inventory management

## Skills Practiced

- Data cleaning and validation  
- Excel formulas: IF, ROUND, structured references  
- Pivot tables and filtering  
- Basic charting and visualization  
- Retail analytics with a focus on fashion

## Why This Project

I created this project to combine my growing Excel skills with my strong interest in fashion. It simulates real-world retail analysis, giving insights into sales, returns, and pricing trends in the fashion industry.

## How to Use

1. Open the `Fashion_Boutique_Analysis.xlsx` file in Excel.
2. Explore the table with structured references.
3. Review the `return_status_check`, `amount_disscounted`, and `price_check` columns for data validation.
4. Interact with the PivotTables to explore trends in returns, ratings, discounts, and revenue.
5. Optionally, modify filters, grouping, and charts to gain additional insights.
